# 🛒 Projeto 01 - Desenvolvimento de Layout

## 📌 Sobre o projeto

Este projeto consiste no desenvolvimento de um layout de e-commerce baseado em um protótipo do Figma, utilizando HTML, CSS (Tailwind) e JavaScript.

O objetivo foi reproduzir com fidelidade o design proposto, garantindo responsividade e interações básicas.

---

## 🚀 Tecnologias utilizadas

* HTML5
* Tailwind CSS
* JavaScript
* Swiper.js (carrossel)

---

## 🎯 Funcionalidades

* Layout responsivo (Desktop e Mobile)
* Menu de navegação com versão mobile
* Mega menu
* Carrossel de produtos utilizando Swiper
* Campo de busca funcional:

  * Exibe a mensagem:

    > Você buscou por: "frase/palavra digitada"

---

## 📱 Responsividade

O projeto foi desenvolvido seguindo o layout do Figma para:

* Desktop
* Mobile

---

## 🎨 Layout

O design foi baseado no protótipo do Figma fornecido no desafio, respeitando:

* Cores
* Tipografia
* Espaçamentos
* Estrutura visual

---

## 📂 Como executar

1. Clone o repositório
2. Abra o arquivo `index.html` no navegador

---

## 📌 Observações

Este projeto foi desenvolvido como parte de um desafio técnico para vaga de estágio, com foco em:

* Fidelidade ao layout
* Organização de código
* Boas práticas de front-end

# 🔎 Projeto 02 - Busca de Perfil no GitHub

## 📌 Sobre o projeto

Aplicação desenvolvida em React que permite buscar perfis do GitHub e exibir informações do usuário, seguindo o layout proposto no Figma.

---

## 🚀 Tecnologias utilizadas

* React
* JavaScript (ES6+)
* CSS
* API do GitHub

---

## 🎯 Funcionalidades

* Campo de busca por usuário do GitHub
* Consumo da API pública do GitHub
* Exibição de dados do usuário:

  * Nome
  * Foto de perfil
  * Bio
    
* Tratamento de erros:

  * Exibe mensagem quando o usuário não é encontrado

---

## ✨ Diferenciais

* Loading durante requisição
* Feedback visual ao usuário
* Interface responsiva
---

## 📡 API utilizada

* https://api.github.com/users/{username}

---

## 📱 Responsividade

Aplicação adaptada para diferentes tamanhos de tela, conforme o layout do Figma.

---

## 🎨 Layout

Interface desenvolvida com base no protótipo do Figma, mantendo fidelidade visual.

---

## 📂 Como executar

```bash
# Clone o repositório
git clone https://github.com/guraiken/Desafio-Tecnico-AVANTI.git

# Acesse a pasta
cd ./pesquisador-github-2Desafio

# Instale as dependências
npm install

# Rode o projeto
npm run dev
```

---

## 📌 Observações

Projeto desenvolvido como parte de desafio técnico para vaga de estágio, com foco em:

* Consumo de API
* Manipulação de estado
* Componentização em React
* Experiência do usuário
